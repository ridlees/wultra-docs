Security Warning

Dear valued customer, we terminated mobile banking app due to a security issue.

What happened?

We identified that a foreign execution code was inserted into the running mobile banking application and it tries to modify its behavior.

What is the risk?

In case a foreign code is inserted into the app, for example via "hooking frameworks" or "code injection", application behavior may be altered. This can lead to leakage of sensitive data, including account information, transactions or even your PIN code. Also, a malicious or fake data may be inserted into the application and as a result, payment or consent can be authorized witnout your knowledge.

What could have caused the issue?

Foreign code may have been iserted in the running app because:

  • Your device is jailbroken and some of the installed non-standard apps (such as Cydia Substrate framework) tries to insert foreign code into the application to modify its behavior.
  • You are running an app on unofficial or beta version of mobile operating system that inserts additional code into the apps.

How to fix the issue?

To resolve the issue, make sure that:

  • You use official version of iOS.
  • Your device is not jailbroken (removal of OS protection).
  • You installed mobile app from App Store.
  • You do not run app in simulator or other non-standard runtime environment.
  • You do not actively debug our app using software development tools.
