Security Warning

Dear valued customer, we terminated mobile banking app due to a security issue.

What happened?

We identified that the mobile banking app was launched in non-standard runtime environment, for example in emulator or virtualization tool.

What is the risk?

If an application runs in a non-standard runtime environment, such as emulator or virtualized environment, instead of an official operating system, sensitive data can be stolen from the mobile app. This includes information such as accounts, transactions or even your PIN code. When running in a non-standard environment, malicious software can insert data into the application and as a result, a fraudulent payment or consent approval can happen without your knowledge.

What could have caused the issue?

Application can run in a non-standard runtime environment because:

  • Your device is modified and it uses virtualization or emulation of processes.
  • You are running an app on unofficial or beta version of mobile operating system that uses virtualization to launch apps.
  • Your device is infected with mobile malware that contains rooting framework and uses a virtualized environment or emulation to attack mobile banking app.
  • You actively try run our application on device emulator.

How to fix the issue?

To resolve the issue, make sure that:

  • You use official version of Android OS.
  • Your device is not rooted (removal of OS protection).
  • You installed mobile app from Google Play.
  • You use an official firmware version.
  • You installed a mobile antivirus that identifies potential malware.
  • You do not run app in emulator or other non-standard runtime environment.
